# koa路由自动加载模块

## 安装依赖

```sh
npm install
```

## 运行

```
npm run dev
```

## 测试

运行成功打开下列网址，查看src/routes下文件与下列url的对应关系

http://locahost:3000
http://localhost:3000/demo
http://localhost:3000/demo/info
